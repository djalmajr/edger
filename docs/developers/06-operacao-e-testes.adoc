O `edger-ext-gateway` tambĂ©m aplica polĂ­ticas de borda locais:

  timeout local, remove `Authorization`, cookies e headers hop-by-hop antes do

* Rate limit local em memĂłria: quando configurado, o gateway mantĂ©m um token
