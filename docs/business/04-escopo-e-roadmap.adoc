=== Escopo e roadmap

==== Escopo atual

O estado atual é uma foundation funcional em progresso:

* Fases 1-6 entregues: core, isolamento inicial, worker management,
  orquestrador, auth/gateway e registro estático de extensões.
* Fase 7 em andamento: execução real JS/TS, Wasm, manifests completos, shell,
  cron, observabilidade e hardening.
* Runtime ativo é Rust-only. O adapter Bun histórico foi removido do caminho
  funcional.
* JS/TS já executa via Deno CLI bridge v1.
* Wasm já executa via `wasmtime` com ABI HTTP mínima.
* `stream` e `sse` têm fallback v1 de primeiro chunk; streaming passthrough real
  ainda está pendente.

==== Fora do escopo atual

* Deploy Kubernetes/Helm, cpanel e marketplace.
* Compatibilidade Node/Next.js completa sem adapters.
* Carregamento dinâmico de crates Rust em runtime.
* Substituição completa do Buntime em produção sem matriz de compatibilidade
  verde.
* `deno_core` embutido finalizado para produção.
* WASI com preopen real e ABI Wasm request/response completa.
