= Edger — Documentação

O Edger é um runtime edge em Rust com contratos compatíveis com a visão do
Buntime: workers como unidades de deploy, manifests, namespaces, autenticação,
extensões, supervisão de lifecycle e execução de JS/TS e Wasm por um
orquestrador nativo.

O projeto elimina o antigo caminho de runtime via adapter Bun. O entrypoint
funcional atual é o binário Rust `edger`, servido pelo crate
`edger-orchestrator`. Workers JS/TS executam por uma ponte Deno CLI v1;
workers Wasm executam por `wasmtime`. `deno_core` embutido permanece o alvo de
produção para JS/TS.
