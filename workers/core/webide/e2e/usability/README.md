# Usability reports

Relatórios gerados ao executar individualmente os arquivos de `../flows/` com o
skill `ux-persona` devem ser adicionados aqui como `<flow-id>--<date>.md`.

O catálogo foi descoberto e criado em 2026-07-14. Ele deve ser revisado pelo
responsável do produto antes da primeira execução completa e os fluxos devem ser
executados sequencialmente, nunca em paralelo no mesmo Browser.
