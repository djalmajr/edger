# Story 08.20 runtime evidence — gateway log duration

Date: 2026-06-29

Scope:
- Gateway recent decisions include `durationMs` for short-circuits and for
  requests completed through `on_response`.
- Requests that continue to a worker are updated with final response status and
  duration by matching `requestId`.
- `GET /api/admin/gateway/logs/stats` computes `duration.tracked`,
  `duration.samples`, and `duration.avgMs` from retained decisions.
- Logs/stats still do not serialize request headers, response headers, bodies,
  `Authorization`, `x-api-key`, raw secrets, or worker payloads.
- No histogram, p95/p99, SSE, persisted history, log deletion, rate-limit reset,
  config mutation, external proxy, cache, or distributed rate-limit in this
  slice.
