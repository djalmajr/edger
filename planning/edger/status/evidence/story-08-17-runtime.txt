# Story 08.17 runtime evidence — gateway operational diagnostics

Date: 2026-06-29

Scope:
- `Extension::diagnostics()` now exposes optional pure JSON diagnostics for
  extension inventory.
- `AdminExtensionInfo` serializes `diagnostics` only when an extension provides
  it.
- `GatewayExtension` exposes local counters for `continue`, `preflight`,
  `redirect`, and `rate_limited` decisions.
- Gateway diagnostics keep the latest 100 local decisions.
- Diagnostic entries include request id, method, path, client, decision, status,
  and `rateLimited`.
- Diagnostic entries do not serialize request headers, body, `Authorization`,
  `x-api-key`, or raw secrets.
- `/api/admin/extensions` includes gateway diagnostics for root operators.
- No gateway-specific SSE, persisted history, log filters, avg duration,
  external proxy, cache, or distributed rate-limit in this slice.
