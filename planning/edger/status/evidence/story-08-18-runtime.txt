# Story 08.18 runtime evidence — gateway read-only Admin API

Date: 2026-06-29

Scope:
- Gateway diagnostics now include a safe `config` block.
- `GET /api/admin/gateway/stats` returns the full local gateway diagnostic
  snapshot.
- `GET /api/admin/gateway/config` returns safe read-only gateway config.
- `GET /api/admin/gateway/logs` returns recent gateway decisions with filters:
  `limit`, `rateLimited`, `status`, and `decision`.
- All gateway Admin API endpoints require root credentials.
- Logs/config do not serialize request headers, bodies, `Authorization`,
  `x-api-key`, or raw secrets.
- No SSE, persisted history, log deletion, rate-limit reset, config mutation,
  external proxy, cache, or distributed rate-limit in this slice.
