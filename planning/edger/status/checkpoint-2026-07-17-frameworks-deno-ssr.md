# Checkpoint 2026-07-17: frameworks Deno SSR

Epic 24 concluído para os nove frameworks da lista oficial do Deno Deploy:
Next.js, Astro, Nuxt, SolidStart, TanStack Start, SvelteKit, Fresh, Lume e
Remix. Os caminhos interativos responderam SSR, API/server routes e assets; o
Lume entregou o `_site` pré-gerado. Fresh, Lume e Remix foram instalados pelo
Admin API com HTTP 201. Remix permanece experimental e Lume é intencionalmente
estático. O pacote/download Next comprovou o limite administrativo sem alterar
o data plane.

Detalhes e receitas: `../docs/frameworks-deno-ssr.md` e
`../docs/compat-matrix.md`.
